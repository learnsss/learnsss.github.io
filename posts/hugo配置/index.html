Hugo配置

2020.05.17

安装hogo

去https://github.com/gohugoio/hugo/releases?after=v0.55.6下载适合的版本

安装完成后cmd输入hugo version来验证hugo版本

添加环境变量puch

用Hugo来生成博客

hugo new site myblog  #建立myblog文件夹来储存

下载并设置主题

cd进入博客根目录

$ git clone https://github.com/imzeuk/hugo-theme-zozo themes/zozo

主题最好与下载的hugo版本想匹配，不然后期配置会出错

在本地启动博客

hugo server -t zozo --buildDrafts
hugo server #查看服务器

启动完成便可以用http://localhost:1313/来查看

新建一篇文章

hugo new posts/blog.md #在post文件夹下新建一篇文章，文章的标签 draft需要设置为false

hugo server -t zozo --buildDrafts

个人博客部署到GitHub

新建仓库，名称必须为自己的用户名+github.io

hugo --theme=zozo --baseUrl="https://learnsss.github.io" --buildDrafts

cd进入public 文件夹

git init

git add .

git commit -m"first"

git remote add origin https://github.com/learnsss/learnsss.github.io.git 

git push -u origin master
